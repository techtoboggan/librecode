# LibreCode Brand Guide

## Identity

**Tagline:** Free Software. Modern Code.

**Mascot:** Winged monkey holding a tater (affectionately known as "Tater").
The wings are teal-to-blue gradient matching the logo palette.
The golden amber body represents warmth and approachability.

**Logo:** Flowing "LC" monogram with:
- Teal-to-blue gradient
- Code symbols: `{}`, `</>`
- Binary streams: `010101`
- Birds in flight (freedom/open source)

## Color Palette

### Primary: Teal
The signature color. Used for interactive elements, links, and brand accents.
- Brand teal: `#0D9488`
- Range: `#EFFCF9` (50) to `#042F2E` (900)

### Secondary: Deep Blue
Used for headings, depth, and pairing with teal in gradients.
- Brand blue: `#1E6CA0`
- Range: `#EFF6FF` (50) to `#0A2239` (900)

### Accent: Golden Amber
From the mascot. Used sparingly for highlights, badges, and CTAs.
- Brand amber: `#D4A017`

### Surfaces
- Dark mode background: `#0A0B0D`
- Dark mode card: `#12141A`
- Light mode background: `#F5F0E8` (warm cream)
- Light mode surface: `#FAFAF7`

## Typography

- **Display/Headings:** Inter (or system sans-serif)
- **Body:** Inter
- **Code:** JetBrains Mono / Fira Code

## Usage

- Dark mode is the default context (terminal tool)
- Light mode for docs/marketing sites
- The gradient (teal -> blue) is the hero treatment
- Mascot appears in onboarding, empty states, error pages
- Logo monogram for favicons and small contexts
